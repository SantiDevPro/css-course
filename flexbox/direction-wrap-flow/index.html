Flex Direction, Wrap y Flow

flex-direction define la dirección en la que los elementos flexibles se organizan dentro de un contenedor (row, column, etc.).

flex-wrap controla si los elementos flexibles se ajustan a una nueva línea cuando no caben en una sola fila o columna (nowrap, wrap).

flex-flow es una propiedad abreviada que combina flex-direction y flex-wrap.
