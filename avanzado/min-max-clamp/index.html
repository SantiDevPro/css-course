Min, Max y Clamp

min(): devuelve el valor más pequeño entre los valores especificados, útil para establecer límites mínimos.

max(): devuelve el valor más grande entre los valores especificados, útil para establecer límites máximos.

clamp(): define un valor dentro de un rango, especificando un valor mínimo, preferido y máximo (clamp(min, preferido, max)).
