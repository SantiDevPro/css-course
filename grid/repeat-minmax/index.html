Repeat y Minmax

Repeat: función que permite simplificar la definición de filas o columnas repetitivas en una cuadrícula, especificando el número de repeticiones y el tamaño de cada una.

Minmax: función que define un rango flexible para filas o columnas, estableciendo un tamaño mínimo y máximo (min, max) según el contenido o el espacio disponible.
