Grid Explicito e Implicito

Grid Explícito: define manualmente mediante propiedades como grid-template-rows y grid-template-columns, especificando las filas y columnas de la cuadrícula.

Grid Implícito: genera automáticamente por el navegador cuando los elementos exceden las filas o columnas definidas explícitamente, utilizando propiedades como grid-auto-rows y grid-auto-columns.
